# 毛孩子档案 - 技术支持

## 应用介绍

毛孩子档案是一款帮助宠物主人集中管理宠物健康记录的应用。

## 功能特性

- 🐾 宠物基本信息管理
- 💉 疫苗接种记录
- 🐛 驱虫记录
- 🏥 就医诊疗记录
- 🧴 日常护理记录
- 📊 健康报告生成
- 📄 PDF导出

## 常见问题

### Q: 如何添加新宠物？
A: 在首页点击 "+" 按钮，填写宠物信息即可。

### Q: 数据会备份到云端吗？
A: 目前数据只存储在本地设备上，建议定期备份。

### Q: 如何导出健康报告？
A: 在宠物详情页点击"健康报告"，然后点击"生成PDF"。
